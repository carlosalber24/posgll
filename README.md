# First Time Setup
```sh
$ sudo apt-get install nodejs
$ sudo apt-get install npm
$ sudo apt-get install git
$ sudo npm install -g cordova ionic
$ cd ~/ionicbazaar-<xxxx>
$ npm install
$ ionic state reset
$ ionic run android
$ ionic run ios
```
